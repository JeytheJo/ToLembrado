# Tô Lembrado 💊

Aplicativo mobile de gerenciamento de cuidados para idosos, desenvolvido para cuidadores que precisam organizar medicamentos, horários e rotinas de saúde.

## Funcionalidades

- Cadastro de múltiplos perfis de idosos
- Registro de tarefas e medicamentos com horários
- Lembretes por período do dia (Manhã, Tarde, Noite)
- Histórico de tarefas realizadas
- Frequência personalizável por dias da semana
- Dados armazenados localmente no dispositivo

## Tecnologias

- React Native + Expo
- SQLite (expo-sqlite)
- Expo Notifications

## Como rodar
```bash
npm install
npx expo start
```
